# Portafolio Personal

## Descripción

Este proyecto corresponde a un portafolio personal desarrollado con HTML y CSS.

Incluye información sobre mí, mis habilidades y algunos proyectos de ejemplo. También cuenta con una sección de contacto y fue publicado utilizando GitHub Pages.

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub
- GitHub Pages
